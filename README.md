# 주요 기능  
⚡ 플레이어 상호작용
- Interaction Base : 상호작용 공통 멤버 정의
- IHandPickable : 손으로 잡을 수 있는 인터페이스 멤버 정의
- Interaction Implement : Interaction Base 상속, 필요 시 IHandPickable 인터페이스를 다중상속해 각 상호작용 구현
